# Password Manager 🔐

A simple Flask + SQLite based password manager to store and manage your passwords securely.

## Features
- Add website, username, password
- Auto-generate strong passwords with symbols
- View all saved passwords as ********
- Delete passwords
- Clean UI with HTML/CSS

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite3
- **Frontend:** HTML, CSS, Jinja2

## Run Locally
1. Clone the repo
   ```bash
   git clone https://github.com/tera-username/password-manager.git
   cd password-manager
2. Install dependencies
     pip install -r requirements.txt
3. Run the app
     python password_manager.py
4. Open 127.0.0.1:5000
